# Editorial Portfolio: A Product-Driven Personal Website

## 0. Project Overview
This is a high-end, editorial-style personal portfolio website designed for **PM / Marketing PM internship applications**. 

Unlike a traditional resume, this project is treated as a **well-designed product**. It aims to showcase diverse experiences (AI research, consulting, competitions) through an interactive, clean, and international narrative.

### Core Philosophy
* **Data-Driven Architecture**: Content is decoupled from the UI.
* **Editorial UX**: High-end typography, minimalist grid, and subtle micro-interactions.
* **Scalability**: New projects and thoughts can be added by updating a single data file without touching the layout code.

---

## 1. Tech Stack
* **Frontend**: HTML5, CSS3 (Custom Variables & Grid), JavaScript (Vanilla)
* **Design**: Figma (Auto Layout & Prototyping)
* **Deployment**: Vercel (CI/CD via GitHub)
* **Interactive Components**: SVG-based World Map & Modal System
